# skillflow
A personal skill tracking web app for students and self-learners.

## Features
- Skill tracking
- Time tracking
- Progress visualization
- Evidence upload
- Growth timeline
- Dark/light mode

## Problem Statement
Students struggle to track their growth and stay motivated while learning new skills.

## Solution
SkillFlow helps users monitor skill growth, practice consistency, and evidence of improvement in one platform.

## Tech Stack
- React
- Tailwind CSS
- Firebase
- Replit
